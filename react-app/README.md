# To Run

## Install Node.js and Node Modules
Node.js can be downloaded from your browser

Install node modules by running the following command in the console:

```shell
npm install
```

## Running Styleguidist

```shell
npm run style
```

## Running Next.js Server

```shell
npm run server
```

## Running Tests

```shell
npm run test
```
OR
```shell
jest
```
